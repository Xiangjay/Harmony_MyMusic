## 我的音乐APP项目

### 项目说明
1. 本项目只为做学习展示，不用做任何商业用途
2. 需要配合后端使用
3. 后端接口用的是别人之前写好的，由于版权问题不便公开，感兴趣可以私信(vx:15241138801)
4. 由于官方推荐的Navigation不能在预览器中运行，为了开发和调试方便这里路由跳转还是用之前的Router

### 项目特色
1. 模拟QQ和网易云音乐做的一个音乐APP
2. 实现了主题的切换
3. 首页里用到了
4. 封装了媒体查询，AVPlayer，用户首选项，HTTP等工具库
